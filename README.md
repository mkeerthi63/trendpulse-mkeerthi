# 📊 TrendPulse: What's Actually Trending Right Now

TrendPulse is a complete data pipeline project that collects, cleans, analyzes, and visualizes trending data using Python.

---

## 🚀 Project Overview

This project is divided into 4 tasks:

1. **Data Collection**
   - Fetch top stories from HackerNews API
   - Extract required fields
   - Save as JSON

2. **Data Processing**
   - Load JSON data using Pandas
   - Clean and preprocess data
   - Save as CSV

3. **Data Analysis**
   - Perform analysis using Pandas & NumPy
   - Compute statistics
   - Add new columns like engagement and popularity

4. **Data Visualization**
   - Create charts using Matplotlib
   - Generate a dashboard with multiple visualizations
